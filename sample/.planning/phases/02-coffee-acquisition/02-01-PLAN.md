# Plan 02-01: Locate and Operate the Coffee Machine

## Goal

The robot identifies the coffee machine, selects a brew, and starts it.

## Tasks

1. [x] Detect the machine with the vision model
2. [x] Press the brew button with the manipulator arm
3. [x] Confirm brewing started (audio + heat signature)

## Notes

Button press force calibrated to 2.5 N — see [[02-RESEARCH]] for why the
first attempt cracked the bezel.

```rust
fn press_button(arm: &mut Arm, target: Point) -> Result<(), ArmError> {
    arm.move_to(target.offset_z(APPROACH_MM))?;
    arm.press(Force::newtons(2.5))
}
```

## Acceptance

- Machine located from any kitchen entry angle
- Brew starts on first press in 9/10 attempts
