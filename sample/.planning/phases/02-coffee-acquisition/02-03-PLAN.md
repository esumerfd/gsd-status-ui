# Plan 02-03: Spill Recovery and Retry Loop

## Goal

When a spill or failed brew is detected, the robot cleans up, reports the
incident, and retries up to two times before asking a human for help.

## Tasks

1. [ ] Detect spills on the counter within 3 seconds
2. [ ] Deploy the absorbent pad and wipe pattern
3. [ ] Log the incident with photos to the run record
4. [ ] Retry the brew; escalate to a human after 2 failures

## Acceptance

- Simulated spill cleaned to sensor-dry in under 60 seconds
- Escalation message includes photo, timestamp, and failure classification
